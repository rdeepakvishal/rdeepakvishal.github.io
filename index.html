01 About

I'm a Business Intelligence Engineer at Amazon Prime Video, where I build the data systems and AI agents that help leaders make faster, sharper decisions. 02 Featured Project

InsightAgent

An agentic BI assistant.

InsightAgent is a production-style agentic BI assistant. You ask a business question in plain English, and the agent writes its own SQL, queries a multi-table warehouse, and returns a narrative answer, a chart, and the exact queries it ran, so nothing is a black box.

It's a public, shareable distillation of an internal agentic BI system I shipped at Amazon Prime Video, where senior product leaders and directors query 100+ tables and 120+ metrics in natural language. This version recreates the core pattern on a synthetic subscription-payments dataset so the whole approach can be inspected end to end.

  • Agentic loop, not one-shot text-to-SQL: queries, reads results, and refines like an analyst would
  • Grounded in a semantic layer so metric definitions stay consistent
  • Read-only safety enforced at two independent layers
  • Full transparency: every answer ships with the SQL that produced it
  • Closed-loop thumbs up/down feedback for continuous improvement
